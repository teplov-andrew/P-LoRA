# ImageEdit

`train.sh` - общее обучение (просто как шаблон)  
`generate_samples.py` - параллельная генерация изображений по чекпоинтам на нескольких GPU. Основной файл для создания генераций  
`distributions_papers.ipynb` - основной файл, где общие графики editability/reconstruction score по всем конфигурациям и режимам  
`Images.ipynb` - сборка дополнительных визуализаций
